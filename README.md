### EDA_Airbnb_Gujarat
This project extracts Airbnb listings from Ahmedabad, Gujarat using Selenium and BeautifulSoup, then performs full EDA on pricing, reviews, property types, and host factors.
### Project Workflow 
1. Web Scraping using Beautiful Soup and selenium 
2. Data Extraction For each listing card, extracted essential columns 
3. Deriving columns using Regex and string processing
4. Data Cleaning using pandas 
5. Exploratory data analysis and visualization - univariate , bivariate and multivariate using matplotlib and seaborn 
6. Applied BoW and NLP for identifying most common words and visualized using WordCloud.

### Conclusion/Insights
1. Ahmedabad and Surat contribute the highest number of Airbnb listings in Gujarat.
2. Most properties are apartments, indicating limited diversity in other accommodation types.
3. Pricing is mainly concentrated in the affordable to mid-range category.
4. Higher-priced or luxury stays are primarily found in Ahmedabad and Gandhinagar.
5. Average ratings remain consistently high across most property types.
6. Hosts with badges receive more reviews and engagement compared to non-badge hosts.
7. Urban areas are well represented, while rural locations have fewer listings.
