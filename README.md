### EDA_Airbnb_Gujarat
### This project extracts Airbnb listings from Ahmedabad, Gujarat using Selenium and BeautifulSoup, then performs full EDA on pricing, reviews, property types, and host factors.
### Project Workflow 
### 1. Web Scraping using Beautiful Soup and selenium 
### 2. For each listing card, extracted 3 columns 
### 3. Deriving columns using Regex and string processing
### 4. Data Cleaning using pandas 
### 5. Exploratory data analysis - univariate , bivariate and multivariate using matplotlib and seaborn
